SimpleSchedule Demo

This page demonstrates a bit of simple-schedule's functionality.

The library is attached to the window object on this page as SimpleSchedule.

Define a schedule with an object

The schedule can be set up with an object with day names as keys. You can use full or abbreviated (three-letter) names, or even mix and match them if that's your fancy.

The times are set by an open and close property, which supports limited parsing. Open times are assumed to be AM, and close times are assumed to be PM. An explicit "am" or "pm" in the time will override this.

For a day that is open 24 hours, you can supply {close: false}.

If you want to be closed all day, use {open: false} or just straight-up false.

Define a schedule with an array

The schedule can also be set up with an array, where the index of the array corresponds to a day's index in the week (starting from 0: Sunday, to 6: Saturday).

Missing an index? If an index is missing, that day will default to being closed all day.
